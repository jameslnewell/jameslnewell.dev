# jameslnewell.me

Repository for [jameslnewell.me](https://jameslnewell.me).

## Installation

```bash
npm install
```

## Building

### For development

1. Run `npm run serve`
2. Browse to `http://localhost:3000`
3. Turn on Live Reload

### For production

1. Run `npm run build`
2. Upload the files in `./dist`
